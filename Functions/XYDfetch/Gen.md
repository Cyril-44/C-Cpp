# 📘 题目：平行线

## 🧾 题目描述

小 C 学会了一个新技能，他想要给小 H 展示。

小 H 最初有一个 $1\sim n$ 的排列 $a_1,a_2,\dots,a_n$。小 C 会进行若干轮操作直到排列已经排好序：

对于 $i$ 从 $1$ 到 $n-k+1$，如果 $a_i,a_{i+1},\dots,a_{i+k-1}$ 不是升序，小 C 就会使用一次新技能将这个子段排序。

不难发现，$k=2$ 的时候就是冒泡排序。但是小 C 操作的太快了，小 H 并没有看清他操作的过程。她怀疑小 C 偷偷作弊了，所以她要求小 C 说出来他使用了多少次新技能。

小 C 立刻就回答出来了，但是小 H 不知道他的答案是否正确，于是她来问你。


## 📥 输入格式
从文件 `maze.in` 读入数据。
**本题有多组测试数据**。

第一行一个正整数 $T$ 表示数据组数。

对于每组数据：

第一行两个整数 $n,k$。

第二行 $n$ 个整数 $a_1,a_2,\dots,a_n$。


## 📤 输出格式
输出到文件 `maze.out` 中。

一个整数表示答案。


## 📌 约束条件

$1\le T\le 10,1\le k\le n\le 2\times 10^5$，$k\ge 2$，保证 $a$ 是 $1\sim n$ 的排列。

| 测试点编号 | 特殊性质 |
|:--------:|:-------:|
| $1\sim2$ | $n\le 200$ |
| $3\sim5$ | $n\le 2000$ |
| $6\sim9$ | $k=2$ |
| $10\sim12$ | $k>\frac n2$ |
| $13\sim16$ | $\forall 1\le i\le k,a_i=i$ |
| $17\sim20$ | 无 |

## 🧪 样例
### 样例 1
**输入：**
```
3

1 3 4 2
6 3
5 1 6 4 3 2
20 7
10 17 8 1 16 13 14 5 20 19 4 15 18 3 11 2 12 9 7 6

```
**输出：**
```
2
6
23

```
