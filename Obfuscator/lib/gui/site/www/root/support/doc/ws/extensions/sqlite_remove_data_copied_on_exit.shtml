Introduction

This "virtual" extension, if enabled, erases SQLite database files copied from CDROM during Stunnix Perl Web Server shutdown. If SQLite database files were not copied from CDROM (as set by option copy_from_hdd of sqlite_data extension), no attempt will be made to delete database files.

Details

No options can be passed to this extension using /site/config/extlist.conf file.

No implementation for this extension is necessary, this extension is "virtual".
