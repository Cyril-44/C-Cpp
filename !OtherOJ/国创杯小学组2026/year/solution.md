这题正解有非常多的实现方法，这里给出一种较为简洁的实现：

1. 考虑第 $m-1$ 年是第 $1$ 年起的第几个闰年，令计算出来的答案为 $n_0$，然后 $n \gets n + n_0$。
2. 计算第 $1$ 年起，第 $n$ 个闰年是多少年。

对于第 2 步的具体说明：

1. 先算出 $c$ 整段的年份，即，考虑到前 $n$ 个闰年中的最后一个满足被 $c$ 整除的年份。如果第 $n$ 个闰年就是这样的年份，直接中断流程。
2. 再算出被 $b$ 跳过的年份，即，考虑到前 $n$ 个闰年中最后一个满足被 $a$ 整除且不被 $b$ 整除的年份。
3. 最后算出 $a$ 整段的年份，即，考虑到第 $n$ 个闰年。

可以证明，在题目条件下，最终答案以及中间的计算值不可能超过 `long long` 类型所表示的范围。

全程在线，时空复杂度均为 $\mathcal O(1)$。

测试点 $1\sim5, 22$ 留给聪明的小朋友。

测试点 $6\sim10$ 留给全域枚举的小朋友。

测试点 $11\sim18$ 留给能够想到枚举 $a$ 的倍数的小朋友。

测试点 $19\sim21$ 留给会二分答案的小朋友。

测试点 $23\sim25$ 留给会正解的小朋友。

~~不难发现正解最短最好写。~~

```cpp
#include <cstdio>
int main() {
    int C, t, a, b, c;
    scanf("%d%d", &C, &t);
    long long m, n;
    for (int i = 1; i <= t; i++) {
        scanf("%lld%d%d%d%lld", &m, &a, &b, &c, &n);
        --m; n += m/a - m/b + m/c;
        c /= b, b /= a;
        long long ans = a * (n / (b*c-c+1)) * (b*c);
        n %= b * c - c + 1;
        if (n) {
            ans += a * ((n-1) / (b-1)) * b;
            n = (n-1) % (b-1) + 1;
            ans += a * n;
        }
        printf("%lld\n", ans);
    }
}
```
