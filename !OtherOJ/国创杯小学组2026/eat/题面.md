## 题目描述

小可可买来了一块长条状巧克力，将其横过来看，从左到右一共有 $n$ 格，每一格巧克力都有一个初始美味度 $a_i$。

小可可计划慢慢吃掉它。她每天都有一个开心值 $p$，然后她会将剩余的巧克力的美味度从左至右顺时针首尾相接写成一个环，并计算第 $i$ 格巧克力与顺时针走 $p$ 格所到达的巧克力的初始美味度的乘积，作为第 $i$ 格巧克力格的美味指数 $b_i$。
|图例|解释|
|:-:|:-:|
|![drawing.svg](asset://4e241916-dbfc-415e-b961-af703c7a9d34)|某一天巧克力的初始美味度可以被描述为 $\{9, 4, 2, 5, 6\}$。    当 $p=2$ 时，当天巧克力的美味指数就是 $\{18, 20, 12, 45, 24\}$。|

小可可会在当天所有的美味指数中选一个 $x$，并确认当天美味指数为 $x$ 的格**之一**为幸运格，然后从左边或者右边一格一格吃巧克力，直到**吃完**幸运格。显然，她每天有许多种吃法，但是，小可可希望每天吃尽可能少的巧克力，所以在这些吃法中，小可可一定会选择一个格数最少的吃法。特别地，如果小可可当天有两种吃法都满足格数最少，她优先选择从左边开始吃。

小可可记录了前 $m$ 天中，她每天的开心值 $p$，以及她选出的那个美味指数 $x$，但是她已经忘了巧克力是怎样吃掉的了。请你帮助她回忆起她是如何吃光巧克力的吧！

## 输入格式

从文件 _eat.in_ 读取数据。

第一行一个正整数 $C$ 表示测试点编号。对于样例 1 满足 $C=0$。

第二行两个正整数 $n,m$。

接下来一行 $n$ 个正整数，第 $i$ 个正整数表示第 $i$ 个巧克力的美味度 $a_i$。

接下来 $m$ 行，每行两个正整数 $p,x$，表意如题。

## 输出格式

输出到文件 _eat.out_。

共 $m$ 行，每行为一个字符和一个数字，中间以空格隔开。其中前面的字符只可以为 `L` 或 `R`，表示这一次是从左边或右边开始吃的；后面的数字表示这一次一共吃了几格巧克力。

## 样例 1 输入

```
6 3
2 3 4 3 2 3
2 9
1 12
4 6
```

## 样例 1 输出

```
R 1
L 2
L 2
```

## 样例 1 解释

该样例中，第一天巧克力的美味指数是 $\{8,9,8,9,4,9\}$，幸运格是 $2,4,6$，一共有 $6$ 种吃法。在这 $6$ 种吃法中，从右往左吃到第 $6$ 格巧克力是吃得最少的吃法。

## 样例 2 输入

```
4 4 
1 2 1 2
1 2
1 2
1 2
1 4
```

## 样例 2 输出

```
L 1
L 1
L 1
L 1
```

## 样例 2 解释

该样例中，第一天巧克力的美味指数为 $\{2,2,2,2\}$，第二天的美味指数为 $\{2,2,2\}$，第三天的美味指数为 $\{2,2\}$，第四天的美味指数为 $\{4\}$。

## 样例 $3 \sim 7$

见选手目录下的 _eat/eat\*.in_ 与 _eat/eat\*.ans_，其中 _\*_ 属于 $\{\tt3,\tt4,\tt5,\tt6,\tt7,\tt8\}$。

第 3 组样例满足 $C=2$，即第 $2$ 个测试点的限制条件。

第 4 组样例满足 $C=3$，即第 $3\sim 4$ 个测试点的限制条件。

第 5 组样例满足 $C=5$，即第 $5$ 个测试点的限制条件。

第 6 组样例满足 $C=6$，即第 $6\sim 8$ 个测试点的限制条件。

第 7 组样例满足 $C=9$，即第 $9\sim 12$ 个测试点的限制条件。

第 8 组样例满足 $C=13$，即第 $13\sim 20$ 个测试点的限制条件。

## 数据范围

对于所有测试数据，均有：
- $1 \le n \le 10^6$；
- $1 \le m \le 10^6$；
- $1 \le x \le 10^{18}$；
- $1 \le t \le n$；
- 对于所有 $i = 1, 2, \cdots, n$，均有 $1 \le a_i \le 10^9$。

| 测试点编号 | $n \le$ | $q \le$ | 特殊性质 |
|:-:|:-:|:-:|:-:|
| $1$         | $10$    | <      | 无 |
| $2$         | $10^3$  | $10^3$ | ^ |
| $3,4$       | $10^5$  | ^      | ^ |
| $5$         | $10^6$  | <      | A |
| $6\sim8$    | ^       | <      | B |
| $9\sim12$   | ^       | <      | C |
| $13\sim 20$ | ^       | <      | 无 |

特殊性质 A：所有 $a_i$ 均相等。

特殊性质 B：在最左边或者最右边 $10$ 块巧克力中，总是存在一块可行的巧克力。

特殊性质 C：对于所有询问，均有 $t=0$。
