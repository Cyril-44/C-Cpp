## 题目描述

小可可有 $n$ 块巧克力排成一排，从左到右编号 $0$ 到 $n-1$，每个巧克力都有一个美味度 $a_i$。

老师会发出 $q$ 次指令，每次给出两个正整数 $x,t$，定义第 $i$ 个巧克力的契合度为 $a_i \times a_{(i+t)\bmod n}$，你可以选择任意一个满足契合度为 $x$ 的巧克力，吃掉在这个位置左边或者右边的所有巧克力（包含当前你选择的巧克力）。但是老师只让你每次吃掉最少个数的巧克力，也就是说你要在所有选择吃巧克力的方案中选择吃掉巧克力个数最小的那一段。如果有多个吃掉个数最少的吃巧克力方案，优先吃掉最靠左的那一段巧克力。**保证至少有一个契合度为 $x$ 的巧克力**。每次指令请你输出你吃掉的这一段的巧克力的左端点标号和右端点标号。

注意：**每次指令之后，剩下的巧克力仍然从 $0$ 开始依次标号。**

## 输入格式

从文件 _eat.in_ 读取数据。

第一行两个正整数 $n,q$。

接下来一行 $n$ 个正整数，第 $i$ 个正整数表示第 $i$ 个巧克力的美味度 $a_i$。

接下来 $q$ 行，每行两个正整数 $x,t$。表示指令的两个参数。

## 输出格式

输出到文件 _eat.out_。

共 $q$ 行，对于每个指令，输出一行两个正整数，表示你吃掉的这一段的巧克力的左端点标号和右端点标号。

## 样例 1 输入

```
6 3
2 3 4 3 2 3
9 2
12 1
6 4
```

## 样例 1 输出

```
5 5
0 1
0 1
```

## 样例 2 输入

```
4 4 
1 2 1 2
2 1
2 1
2 1
4 1
```

## 样例 2 输出

```
0 0
0 0
0 0
0 0
```

## 数据范围

对于所有测试数据，均有：
- $1 \le n \le 10^6$；
- $1 \le m \le 10^6$；
- $1 \le x \le 10^{18}$；
- $1 \le t \le n$；
- 对于所有 $i = 1, 2, \cdots, n$，均有 $1 \le a_i \le 10^9$。

| 测试点编号 | $n \le$ | $q \le$ | 特殊性质 |
|:-:|:-:|:-:|:-:|
| $1$         | $10$    | <      | 无 |
| $2$         | $10^3$  | $10^3$ | ^ |
| $3,4$       | $10^5$  | ^      | ^ |
| $5$         | $10^6$  | <      | A |
| $6\sim8$    | ^       | <      | B |
| $9\sim12$   | ^       | <      | C |
| $13\sim 20$ | ^       | <      | 无 |

特殊性质 A：所有 $a_i$ 均相等。

特殊性质 B：在最左边或者最右边 $10$ 块巧克力中，总是存在一块可行的巧克力。

特殊性质 C：对于所有询问，均有 $t=0$。
