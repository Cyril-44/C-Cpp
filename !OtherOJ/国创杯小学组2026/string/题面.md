## 题目描述

快要小学毕业的小可可决定给好朋友小果果送毕业礼物。

现在小可可有一个长度为 $n$ 的小写字符串 $S$。

定义 $S_i$ 表示字符串 $S$ 的第 $i$ 个字母。**“好串串”是指前一半字典序单调不降、后一半字典序单调不升的回文串。** 即长度为 $m$ 的“好串串”满足以下几个条件：
- $S_1\le S_2\le\cdots\le S_{\lceil \frac{m}{2}\rceil}$
- $S_{\lceil \frac{m}{2}\rceil} \ge S_{\lceil \frac{m}{2}\rceil-1} \ge \cdots\ge S_m$
- 对于所有 $i=1,2,\cdots,m$，满足$S_{i}=S_{m-i+1}$。

满足 $S_i \lt S_j$ 当且仅当 $S_i$ 在字母表中排在 $S_j$ 前面，$\lceil \frac{m}{2}\rceil$ 是指 $\frac{m}{2}$ 上取整。

如 $\texttt{z}$，$\texttt{bbb}$，$\texttt{accgcca}$，$\texttt{ccddeeddcc}$，$\texttt{ghg}$ 等是“好串串”，$\texttt{acbca}$，$\texttt{bbggb}$ 等不是“好串串”。

现在小可可要把 $S$ 分割成若干个不相交的“好串串”送给小果果。因为小果果不想让书包里堆满“好串串”，所以小可可要让分割出的“好串串”个数**尽量少**。

可是小可可不会分割，请你来告诉他**最少**能分割成多少个“好串串”吧！

## 输入格式

本题读入量较大，建议使用 `scanf` 等高效方式进行读入。

注意：我们在选手目录下 _string/string.cpp_ 提供本题读入模板。你可以参考模板进行代码编写，也可以自己编写代码。

**本题多组测试。**

从文件 _string.in_ 中读取数据。

第一行两个正整数 $C, t$，表示测试点编号和数据组数，对于样例 1 满足 $C=0$。

接下来 $t$ 行，每行一个小写字符串 $S$，表示小可可的字符串。

## 输出格式

输出到文件 _string.ans_ 中。

输出包含 $t$ 行，每行一个正整数，表示这组数据的答案。

## 样例 1 输入

```
0 5
czccc
ababa
edffd
bbdddbb
aaababa

```

## 样例 1 输出

```
2
3
2
1
3
```

## 样例 1 解释

该样例共包含五组测试数据。

对于第一组测试数据，可以分割为 $\texttt{czc}$ 和 $\texttt{cc}$。

对于第二组测试数据，可以分割为 $\texttt{a},\texttt{b},\texttt{aba}$ 或 $\texttt{aba},\texttt{b},\texttt{a}$。

对于第三组测试数据，可以分割为 $\texttt{e}$ 和 $\texttt{dffd}$。

对于第四组测试数据，整个串都是好串串。

对于第五组测试数据，可以分割为 $\texttt{aaa},\texttt{b}, \texttt{aba}$。

## 样例 $2\sim8$

见选手目录下的 _string/string\*.in_ 与 _string/string\*.ans_，其中 _\*_ 属于 $\{\tt2,\tt3,\tt4,\tt5,\tt6,\tt7,\tt8\}$。

第 2 组样例满足测试点 $2$ 的限制条件且 $C=2$。

第 3 组样例满足测试点 $3,4$ 的限制条件且 $C=3$。

第 4 组样例满足测试点 $5\sim7$ 的限制条件且 $C=5$。

第 5 组样例满足测试点 $8,9$ 的限制条件且 $C=8$。

第 6 组样例满足测试点 $10\sim12$ 的限制条件且 $C=10$。

第 7 组样例满足测试点 $13\sim15$ 的限制条件且 $C=13$。

第 8 组样例满足测试点 $16\sim19$ 的限制条件且 $C=15$。

## 数据范围

对于所有测试数据，均有：

- $1 \le t \le 10^3$；
- $1 \le n \le 5\times10^6$；
- $1 \le \sum n \le 4\times10^7$；
- 串 $S$ 仅包含小写字母。

| 测试点 | $n \le$ | $\sum n \le$ | 特殊性质 |
|:-:|:-:|:-:|:-:|
| $1$ | $5\times10^6$ | $4\times10^7$ | A |
| $2$ | $10$ | $20$ | 无 |
| $3,4$ | $200$ | $500$ | ^ |
| $5\sim7$ | $5000$ | $5\times10^4$ | ^ |
| $8,9$ | $10^5$ | $2\times10^6$ | B |
| $10\sim12$ | $5\times10^6$ | $4\times10^7$ | ^ |
| $13\sim15$ | ^ | ^ | C |
| $16\sim19$ | $10^5$ | $2\times10^6$ | 无 |
| $20\sim25$ | $5\times10^6$ | $4\times10^7$ | ^ |

特殊性质 A：满足 $S$ 全为字符 $\texttt{a}$。

特殊性质 B：满足 $S$ 中不存在任意两个相邻字母相同。

特殊性质 C：满足 $S$ 随机生成。
