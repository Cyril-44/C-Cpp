## 题目描述

小可可家里来了 $m$ 个小朋友。

小可可家里有 $n$ 个糖果，每个糖果的美味值都为 $x$，她想给每个小朋友分 $k$ 个糖果。因为糖果太多了数不过来，所以她问你，她分完糖果后剩下的糖果美味值之和是多少。

如果小可可的糖果不够分，请你输出 $-1$。

## 输入格式

从文件 _candy.in_ 读取数据。

一行 $4$ 个正整数，分别为 $n,x,m,k$。

## 输出格式

输出到文件 _candy.out_ 中。

输出一行一个整数，表示答案。

## 样例 1 输入

```
9 3 2 3
```

## 样例 1 输出

```
9
```

## 样例 1 解释

该样例中，小可可会剩下 $3$ 个糖果，美味值之和为 $9$。

## 样例 2 输入

```
10 4 5 2
```

## 样例 2 输出

```
0
```

## 样例 2 解释

该样例中，小可可恰好分完所有糖果，美味值之和为 $0$。

## 样例 3 输入

```
9 4 5 2
```

## 样例 3 输出

```
-1
```

## 样例 3 解释

该样例中，小可可的糖果只够分给第二个小朋友 $4$ 颗糖果，输出 $-1$。

## 样例 4

见选手目录下的 _candy/candy4.in_ 与 _candy/candy4.ans_。

该样例满足测试点 $7,8$ 的限制条件。

## 数据范围

对于所有测试数据，均有：

- $n,x,m,k\le 10^6$ 且都为正整数。

|测试点|$n,x,m,k \le$|
|:-:|:-:|
|$1\sim 3$|$10$|
|$4\sim 6$|$1000$|
|$7,8$|$10^4$|
|$9,10$|$10^6$|
