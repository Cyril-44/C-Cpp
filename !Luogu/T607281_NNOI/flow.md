# T607281 【MX-J-T2】流水

## 题目背景

水往低处流。人也是吗？

## 题目描述

众所周知，水是往低处流的。你有一个 $n$ 个池塘组成的树状池塘组，以 $1$ 号池塘为根。每个父池塘和每个子池塘都有一条小溪相连，父池塘高度大于子池塘。

每个池塘填满需要 $1$ 个单位的水，每条小溪填满也需要一个单位的水。如果水想从上往下流，首先必须填满相连的小溪及经过的所有池塘。

这 $n$ 个池塘会轮流受到降水的洗礼。第 $i$ 个池塘会在第 $t_i$ 秒受到 $s_i$ 个单位的降水，所有降水会在一秒内填满需要填满的地方。

你可以在分岔口决定是否流水，可以决定往任意方向流任意单位的水，但是首先自己本身必须已经被填满。

那么请问，最少多少秒**后**，所有**池塘**都将被水填满？

特别的，所有池塘都可以积**任意单位**的水，但是只要有**一个单位**的水我们就定义其被**填满**。

## 输入格式

第一行一个 $n$,含义如题面所示。

后面 $n-1$ 行，每行 $u,v$，描述一条边。

后面跟着 $n$ 行，第 $i$ 行两个数，代表 $t_i,s_i$。

## 输出格式

输出一个数，代表最少多少秒后，所有**池塘**都将被水填满。

## 输入输出样例 #1

### 输入 #1

```
5
1 2
1 3
2 4
2 5
1 2
3 3
4 3
2 1
5 3
```

### 输出 #1

```
4
```

## 说明/提示

#### 本题采用捆绑测试。

| 子任务编号 |特殊性质 | 分值|
| :-----------: | :-----------: |:-----------:|
| $1$ | $1 \le n \le 10$ | $10$|
| $2$ | 保证树是一条链且根节点为端点 |$15$|
| $3$ | $t_1 = n$ |$5$|
| $4$ | $1 \le n \le 1000$ |$20$|
| $5$ | $s_i = 1$ |$5$|
| $6$ | 无特殊性质 |$45$|


对于 $100\%$ 的数据，$1 \le n \le 2 \times 10^5$,$1 \le t_i \le n$ 且所有 $t_i$ 互不相同，$1 \le s_i \le 4 \times 10^5$。

```cpp
#include <bits/stdc++.h>
using namespace std;

vector<int>a[200005];
int n;
int rt[200005];
int t[200005],s[200005];
void dfs(int w,int f,int x){
	rt[w]=1;
	for(int i=0;i<a[w].size();i++){
		if (a[w][i]==f)  continue;
		dfs(a[w][i],w,x);
		if (rt[a[w][i]]==0) continue;
		rt[w]+=(rt[a[w][i]]+1);
	}
	if (t[w]<=x) rt[w]=max(0,rt[w]-s[w]);
}
bool check(int x){
	memset(rt,0,sizeof(rt));
	dfs(1,0,x);
	return rt[1];
}
int main(){
//	freopen(".in","r",stdin);
//	freopen(".out","w",stdout);
	ios::sync_with_stdio(false); 
	cin>>n;
	for(int i=1;i<n;i++){
		int u,v;
		cin>>u>>v;
		a[u].push_back(v);
		a[v].push_back(u);
	}
	for(int i=1;i<=n;i++){
		cin>>t[i]>>s[i];
	}
	int l=1,r=n;
	while(l<r){
		int mid=(l+r+1)/2;
		if(check(mid)) l=mid;
		else r=mid-1; 
	}
	cout<<l+1<<endl;
	return 0;
}

ans: 4
*/
```
