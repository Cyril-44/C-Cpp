# T779654 构造题

## 题目背景

构造！

## 题目描述

我们给你一棵 $n$ 个点的树，分为黑点和白点。每个点有一个 $c_i$，定义 $c_i = 1$ 的点为黑点，$c_i = 0$ 的点为白点。树上的每条边都有一个边权 $w_i$。你需要构造一棵树，包含所有黑点和任意数量的白点。在用点最少的前提下，保证对于任意两个黑点 $(u,v)$ 其最短路径上的边权最大值不变。注意，你构造的这棵树只需要保证点编号不变，不必保证任何树的形态。显然你不会使用超过 $n$ 个点，所以请不要使用 $> n$ 的编号。

## 输入格式

第一行一个 $n$，代表树的点数。

后面 $n-1$ 行，每行三个数 $u_i,v_i,w_i$ 描述一条边。

然后一行 $n$ 个数，第 $i$ 个数代表 $c_i$。

## 输出格式

第一行输出你构造的树的点数 $m$。

后面 $m-1$ 行每行描述一条边。

## 输入输出样例 #1

### 输入 #1

```
5
1 2 3
1 3 4
2 4 2
2 5 4
1 1 0 1 1
```

### 输出 #1

```
4
1 2 3
2 4 2
2 5 4
```

## 说明/提示

注意，如果选手正确输出最少点数可以获得测试点 $30\%$ 的分数。

如果构造出方案可以获得 $100\%$ 的分数。
