# 序列(ball)

## 题目描述

你有一个 $n$ 个数的序列，第 $i$ 个数为 $a_{i}$。

你可以对这个序列进行 $m$ 次操作，每次可以给 $a$
序列中的任何一个数加上一个 $\lbrack - x,x\rbrack$ 内的整数。

小 A 想知道，$m$ 次操作后有多少个不同的整数可能成为这个序列的中位数？

因为这里保证 $n$ 是奇数，所以序列 $a$
的中位数是这个序列按数值从小往大排序后第 $\frac{n + 1}{2}$ 个数。

## 输入格式

第一行三个数，$n$,$m$,$x$。

第二行 $n$ 个数，第 $i$ 个数为 $a_{i}$。

## 输出格式

一个数，输出可能整数的个数。

## 样例输入 1

    5 2 3
    1 2 3 4 5

## 样例输出 1

    5

## 数据范围与提示

对于 $10\%$
的数据，$1 \leq n,m \leq 5$，$1 \leq x \leq 20$，$- 100 \leq a_{i} \leq 100$。

对于 $40\%$
的数据，$1 \leq n \leq 400$，$1 \leq m \leq 800$，$1 \leq x \leq 300$，$- 10^{3} \leq a_{i} \leq 10^{3}$。

对于 $100\%$
的数据，$1 \leq n \leq 10^{6}$，$1 \leq m \leq 10^{8}$，$0 \leq x \leq 10^{10}$，$- 10^{12} \leq a_{i} \leq 10^{12}$，保证
$n$ 为奇数，所有输入均为整数。
