# T740791 Mex

## 题目背景

省选是啥？我问我自己。

## 题目描述

给你一个长度为 $n$ 的 $0 \sim n-1$ 的排列 $a$，你要完成以下两种操作共 $m$ 次：

+ `1 x y` 将 $a_x$ 与 $a_y$ 互换。

+ `2 x y` 求满足 $1 \le l \le r \le n$ 且 $a_l \sim a_r$ 的 mex 在 $x \sim y$ 的 $(l,r)$ 的个数。

## 输入格式

第一行两个数 $n,m$。

第二行 $n$ 个数描述 $a$。

后面 $m$ 行每行描述一个操作。

## 输出格式

对于每个询问操作输出答案。

## 输入输出样例 #1

### 输入 #1

```
5 5
3 1 2 4 0
2 2 3
2 3 3
1 5 3
2 2 2
2 5 5
```

### 输出 #1

```
1
1
4
1
```

## 说明/提示

本题有 $20$ 个等分测试点。

| 测试点  | 特殊性质  |
|:-:|:-:|
|  $1 \sim 3$  |  $1 \le n,m \le 50$ |
| $4 \sim 8$  | $1 \le n,m \le 10^3$  |
| $9 \sim 12$  | 没有第一种操作  |
| $13 \sim 16$  |自始至终满足 $a_1 = 0$  |
| $17 \sim 20$ | 无特殊性质|

对于 $100\%$ 的数据，$1 \le n,m \le 10^6$，保证 $a$ 是一个 $0 \sim n-1$ 的排列。
