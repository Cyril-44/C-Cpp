# Mex

### 题目描述

云小斗一个长度为 $n$ 的 $0 \sim n - 1$ 的排列
$a$，云小斗要完成以下两种操作共 $m$ 次：

- `1 x y` 将 $a_{x}$ 与 $a_{y}$ 互换。

- `2 x y` 求满足 $1 \leq l \leq r \leq n$ 且 $a_{l} \sim a_{r}$ 的 mex
  在 $x \sim y$ 的 $(l,r)$ 的个数。

## 输入格式

第一行两个数 $n,m$。

第二行 $n$ 个数描述 $a$。

后面 $m$ 行每行描述一个操作。

# 输出格式

对于每个询问操作输出答案。

# 样例输入1

    5 5
    3 1 2 4 0
    2 2 3
    2 3 3
    1 5 3
    2 2 2
    2 5 5

# 样例输出1

    1
    1
    4
    1

# 数据范围和约定

本题有 $50$ 个等分测试点。

  |测试点        | 特殊性质 |
  |--------------| -------------------------- |
  |$1 \sim 8$    | $1 \leq n,m \leq 50$ |
  |$9 \sim 20$   | $1 \leq n,m \leq 10^{3}$ |
  |$21 \sim 30$  | 没有第一种操作 |
  |$31 \sim 40$  | 自始至终满足 $a_{1} = 0$ |
  |$41 \sim 50$  | 无特殊性质 |
