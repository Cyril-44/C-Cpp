# T735231 交朋友

## 题目背景

交朋友要注意的事情很多很多。

## 题目描述

有 $n$ 个人是你的备选朋友，第 $i$ 个人有真诚值 $w_i$ 和价值 $v_i$。你会设置一个真诚阀值 $x$ 和一个价值阀值 $y$，那么你能交到的朋友就是 $\sum_{i=1}^{n} [w_i \ge x \land v_i \ge y]$。请你分别求出可以至少交到 $1 \sim n$ 个朋友的阀值 $x+y$ 的最大值。

## 输入格式

第一行一个 $n$，代表人数。

后面 $n$ 行每行两个数，形容一个人。

## 输出格式

一行 $n$ 个数，如题意所示。

## 输入输出样例 #1

### 输入 #1

```
4
4 3
2 5
10 15
17 18

```

### 输出 #1

```
35 25 7 5
```

## 说明/提示

| Subtask  | 特殊性质  |  分值 |
|:-:|:-:|:-:|
| $1$  | $1 \le n \le 100$ |  $5$  |
| $2$  | $1 \le n \le 10^3$ |$15$  |
| $3$  | $1 \le n \le 5 \times 10^3$  | $20$  |
| $4$  | A  |  $5$  |
| $5$  | $1 \le a_i,b_i \le 100$  |  $10$  |
| $6$  | $1 \le a_i,b_i \le 10^5$  |  $20$  |
| $7$  | 无  | $25$   |

特殊性质 A：对于 $1 \le i < n$，满足 $w_i < w_{i+1},v_i < v_{i+1}$。

对于 $100\%$ 的数据，$1 \le n \le 2 \times 10^5,1 \le w_i,v_i \le 10^9$。
