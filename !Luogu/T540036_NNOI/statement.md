# 吵闹班级(noise)

## 题目描述

在一个班中，有 $n$ 个同学，第 $i$ 个同学的吵闹值为 $a_{i}$。

云小斗需要选择一个 $k$，把他们分成 $k$ 组，每组 $\frac{n}{k}$ 个人，第
$i$ 组是第 $(i - 1) \times \frac{n}{k} + 1 \sim i \times \frac{n}{k}$
个人。

我们称一组的吵闹值为这一组所有同学吵闹值的最大值。所有同学的吵闹值为这
$k$ 组的吵闹值之和乘上
$\frac{n}{k}$。云小斗想要知道，班上的吵闹值最小为多少。请你为云小斗输出这个最小值。

## 输入格式

第一行，一个 $n$，代表学生的个数。

第二行 $n$ 个数，第 $i$ 个数代表 $a_{i}$。

## 输出格式

一个数，表示最小吵闹值。

## 样例输入 1

    4
    10 11 14 12

## 样例输出 1

    56

## 数据范围与提示

对于 $10\%$ 的数据，$1 \leq n \leq 10$。

对于 $30\%$ 的数据，$1 \leq n \leq 1000$。

对于 $50\%$ 的数据，$1 \leq n \leq 10^{5}$。

对于另外 $10\%$ 的数据，$a_{i} = 0$。

对于 $100\%$
的数据，$1 \leq n \leq 5 \times 10^{6},0 \leq a_{i} \leq 10^{6}$。
